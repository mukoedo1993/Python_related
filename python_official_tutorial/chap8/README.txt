There are (at least) two distinguishable kinds
of errors: syntax errors and exceptions.

Syntax errors, also known as parsing errors, are perhaps
the most common kind of complaint.


Even if a statement or expression is syntactically correct,
it may cause an error when an attempt is made to execute it.
